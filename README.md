# Weather-app
A Pyhon-based Simple Weather App 
A simple Python application that fetches real-time weather data from the OpenWeatherMap API.  
This project uses the `request` library to make API calls and displays the current temperature, humidity, and weather description for any city.

Technologies Used
- Python 3
- OpenWeatherMap API
- Requests library
